# avik/path

Routing engine for the Avik framework.

## Features
- HTTP method routing
- Route parameters
- Named routes
- Middleware references

## Design Rules
- No execution logic
- No framework glue
- Immutable routes

## Stability
Frozen after v1.0.0

## Requirements
- PHP 8.1+
- avik/seed ^1.0
- avik/flow ^1.0
